RBC Stock Api
---------------------

 * Introduction
 
This project features an API used to perform CRUD opeations on stock records stored in a mySQL database. The API can handle uploading sinlge/multiple records via JSON or file, deleting single records and querying records by stock ticker.
 
 * Requirements
 
JAVA SE 11

* Details

This project uses Spring Boot bean injection to connect the data access layer, service layer and controller layer. Unit tests are written for a single function for each layer. Lombok is used to reduce code in the entity and testing is done with POSTMAN.
